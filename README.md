## Getting Started (Local Development)
Clone the repository and install dependencies:

```bash
git clone https://github.com/alabaganne/Portfolio.git
cd Portfolio
npm install
```

Launch the development server with Turbopack:

```bash
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000) to explore the site. Update `src/app/page.js` and related components to feature new case studies, integrate analytics, or tailor messaging for specific hiring campaigns.

## Deployment on Hostinger VPS
1. Build the production bundle:
   ```bash
   npm run build
   ```
2. Copy the `.next`, `public`, and `package.json` files to your Hostinger VPS (via SSH, Rsync, or CI/CD pipelines).
3. Install production dependencies and start the Node.js server with a process manager such as PM2:
   ```bash
   npm install --omit=dev
   pm2 start npm --name "portfolio" -- start
   ```
4. Configure Nginx or Apache to reverse proxy traffic to the Node.js process and secure the site with Letâ€™s Encrypt SSL certificates.
5. Automate deployments with GitHub Actions or your preferred CI/CD solution to maintain uptime and consistent releases.
